# 🚗 Car Price Prediction App

A Machine Learning powered web application that predicts the resale value of used cars based on various car details.

## 📌 Features

* Predict used car resale price instantly
* Clean and attractive Streamlit UI
* Dynamic car company and model selection
* Uses trained Machine Learning model
* Responsive dashboard design

---

## 🛠 Technologies Used

* Python
* Streamlit
* Pandas
* Pickle
* Machine Learning

---

## 📂 Project Structure

```bash
Car-Price-Prediction/
│
├── App.py              # Main Streamlit application
├── model.pkl           # Trained ML model
├── final.csv           # Dataset used for dropdown values
├── README.md           # Project documentation
├── requirements.txt    # Required libraries
```

---

## 📊 Input Features

The model uses the following inputs:

* Company Name
* Car Name
* Manufacturing Year
* Kilometers Driven
* Fuel Type

---

## ⚙ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/car-price-prediction.git
```

Move to project folder:

```bash
cd car-price-prediction
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶ Run Application

Start the Streamlit app using:

```bash
python -m streamlit run App.py
```

or

```bash
streamlit run App.py
```

---

## 📦 Requirements

Create a `requirements.txt` file with:

```txt
streamlit
pandas
numpy
scikit-learn
matplotlib
joblib
```

Install:

```bash
pip install -r requirements.txt
```
